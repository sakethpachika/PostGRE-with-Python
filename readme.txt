Fake Date is generated using python faker library in combination with couple of csv files as input to generate logical data

Python jupyter notebook Data_Generator can be used to create database and also load the data into database

further 6 csv files are created using the jupyter notebook

create.sql file is used to create a healthcare database and all the relevant tables are generated in the database with necessary constraints, used psql terminal to execute the file

load.sql loads the csv files generated using jupyter notebook into the healthcare database, the path is absolute, please make sure to update the path while running this file
